Bootcamp 2021: Snake Game

My initial project at City, that I learnt to create from the lectures given during bootcamp, was the retro game Snake. The project utilises arrays, position vectors and sound and image files to produce an authentic experience. The game has the snake move using keyPressed and aligns the food and snake with an ArrayList which stores boxes of size 20 in the sketch. Also, the boolean variable lose declares when the game ends - this could be from the snake hitting any side of sketch or the front of the snake colliding with itself. Finally, once the turn ends, the score is displayed as an initial highscore, and can be beaten everytime the user restarts a game.
